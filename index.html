BTAS is a C++ software framework for tensor computation. It is both a standardized tensor concept specification and an efficient reference implementation.

Features include:

  • Dense Tensor class with efficient BLAS-like routines.
  • Tensor slicing and other views.
  • Block-sparse tensor class and quantum numbers.

Quick link to draft tensor interface specification.

Contracting Tensors

// Set index dimensions (tensor shape)
Tensor<double> A(10,2,10,2),
               B(2,10,2);


// Access tensor components (0-indexed)
A(0,1,0,1) = -0.5;
A(1,1,1,1) = 0.5;
B(0,4,1)   = 1.8;
// etc.

// Use integers (obtained from an enum in this example)
// to specify uncontracted and contracted indices
enum { i,j,k,l,m,p };
// result will be C = 2.0*A*B
contract(2.0,
         A,{i,j,k,l},
         B,{m,k,p},
         0.0,
         C,{m,l,i,j,p});
